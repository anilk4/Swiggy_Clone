# Swiggy App

Welcome to the Swiggy App project! This project is a React-based web application that replicates the functionality of the Swiggy food delivery service. It's built from scratch without using create-react-app, utilizing Parcel as a bundler, Redux for state management, and several other technologies and best practices.

## Features

- **Custom React App**: Built a React app without using create-react-app for more flexibility and customization.

- **Parcel Bundler**: Utilized Parcel as the bundler for efficient and fast build processes.

- **Redux State Management**: Implemented Redux for efficient state management across the application.

- **Shimmer Effect**: Incorporated a shimmer effect for the initial rendering of components, providing a smooth loading experience.

- **Real-time Swiggy API Integration**: Used a real-time Swiggy API to fetch and display relevant data in the application.

- **Lazy Loading and Code Splitting**: Implemented lazy loading and code splitting for better performance and reduced initial load times.

- **Tailwind CSS**: Styled the application using Tailwind CSS for a clean and responsive design.

- **Custom Hooks**: Implemented custom hooks to ensure cleaner, more modular, and efficient code.

- **Optional Chaining Concept**: Utilized optional chaining to avoid application crashes due to undefined values.

- **CORS Error Handling**: Implemented handling for CORS errors to ensure smooth communication with external APIs.
